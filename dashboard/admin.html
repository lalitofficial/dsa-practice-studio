Help Center

Full documentation for every feature, workflow, and best practice.

Getting started

Set up a sheet, import data, and begin tracking progress.

  • Create a sheet in Sheet Management.
  • Import CSV/XLSX from the Admin or dashboard Import button.
  • Select the sheet from the header tabs.

Daily workflow

Use this flow to stay consistent and focused.

  • Filter by unit/chapter to narrow scope.
  • Mark done when you complete a question.
  • Add notes for patterns or mistakes.
  • Star questions you want to revisit.
  • Review notes and starred items in Revision.

Dashboard essentials

How core learner tools behave.

  • Search matches title or id keywords.
  • Filters stack (unit, chapter, status, difficulty, starred).
  • Notes modal opens per-question notes.
  • Links show LeetCode/YT/Web; grey means missing.
  • Progress strip updates for the current filter scope.

Tracking panel

What each tracker metric means.

  • Overall completion totals done across all sheets.
  • Focus cues summarize starred, notes, and todo.
  • Difficulty mix aggregates difficulty tags.
  • Momentum shows solved count over the last 7 days.
  • Practice heatmap counts only done questions for the active sheet.
  • Sheet health lists gaps (missing links/difficulty).

Sheet management

Create, edit, and maintain sheets confidently.

  • Create sheet adds a new sheet entry.
  • Save all names applies edited labels at once.
  • Bulk actions reset progress, clear notes, or delete selected sheets.
  • Duplicate clones a sheet and its progress.
  • Export downloads a sheet as CSV/JSON.

Unit management

Refactor naming or difficulty without re-importing.

  • Rename updates unit or chapter across a sheet.
  • Bulk difficulty fills unit/chapter difficulty in one pass.
  • Only fill missing protects existing tags.

UI management

Control naming, layout, theme, and link behaviors.

  • Labels rename Question/Unit/Chapter across the UI.
  • Header visibility toggles Import/Export/Search/Filters/Last sync.
  • Layout controls adjust compact mode and column visibility.
  • Theme colors override accent, surface, and background.
  • Default links generate search URLs for missing links.
  • Settings transfer export/import UI settings as JSON.

Revision & notes

Centralize review of learning feedback.

  • Revision panel lists all notes and starred items.
  • Notes hub aggregates question notes from the dashboard.
  • Starred filter shows only important questions.

Import format

CSV/Excel columns (only first three required).

  • 1) Unit name
  • 2) Chapter name
  • 3) Question name
  • 4) LeetCode link (optional)
  • 5) YouTube link (optional)
  • 6) Default note (optional)
  • 7) Difficulty (optional)
  • 8) Starred (optional)

Troubleshooting

Health checks and recovery tools.

  • Site health shows storage paths and core file status.
  • Diagnostics exports JSON with insights + health.
  • Reset UI restores theme and layout defaults.
  • Reset view clears saved filters and scroll position.
