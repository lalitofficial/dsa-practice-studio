Overview

Clock, timer, and stopwatch are the only widgets available today. They float independently and can be dragged anywhere on the dashboard while the launcher button keeps the dialog tucked away when you are in the flow.

  • Clock: always-on timestamp.
  • Timer: countdown helper with an inline preset.
  • Stopwatch: track elapsed coding or review time.

Use the floating W button to open the widget dialog; every widget you enable will show up in the draggable overlay and stay synced with the toolbar controls.

How to control widgets

Toggle each helper directly inside the widget dialog. The same dialog lets you pick a timer preset, toggle the floating overlay, and head to the Widget Studio page for documentation or future updates.

Once the overlay is visible, drag the W launcher anywhere it feels comfortable. Each card keeps its last position, so you can build a custom layout of clock/timer/stopwatch widgets and leave them running.
